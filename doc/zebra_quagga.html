一、zebra安装

1、编译安装

vim ./lib/zebra.h +103

增加：

#ifdef GNU_LINUX

#define  __USE_GNU

#endif

./configure

sudo make

sudo make install

2、配置zebra运行环境

因为系统中有简单的例子，于是就用现有的配置文件，配置登陆和进入特权模式的密码，主机名等。

sudo cp zebra.conf.sample /usr/local/etc/zebra.conf

配置vtysh

sudo cp vtysh.conf.sample /usr/local/etc/vtysh.conf

3、启动zebra demo程序

./zebra –d

4、远程登陆

telnet 127.0.0.1 2601 

二、quagga安装

1、

./configure --enable-vtysh

--enable-user=root

--enable-group=root

--enable-vty-group=root

出现问题，提示少了libreadlin,解决方法：

sudo apt-get install libreadline5-dev

安装完成之后再重新用配置命令

./configure --enable-vtysh

--enable-user=root

--enable-group=root

--enable-vty-group=root

之后编译与安装

make

make install

2、使用zebra自带的样例作为其启动的配置文件。

cd zebra

sudo cp ./zebra.conf.sample /usr/local/etc/zebra.conf

3、启动zebra

sudo ./zebra –d

4、远程登录

telnet 127.0.0.1 2601

5、如果想启动rip

cd ripd

sudo cp ./ripd/ripd.conf.sample /usr/local/etc/ripd.conf

sudo ./ripd -d

telnet 127.0.0.1 2602
